Slinky — Slurm and Kubernetes

Overview

Slinky™ is SchedMD's set of projects to enable interoperability between Slurm and Kubernetes.

Links

The SlinkyProject on GitHub hosts the associated Slinky components:

  • The Slurm Operator provides for management and auto-scaling of Slurm clusters running as Kubernetes pods: slurm-operator
  • Slurm REST client Go library, shared by the Slinky projects: slurm-client
  • Slurm Prometheus exporter, used to gather metrics for HPA (Horizontal Pod Autoscaling) and Grafana: slurm-exporter
  • Docker files for the associated container images: containers
